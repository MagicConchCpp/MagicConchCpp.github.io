int main (int argc, char *argv[])中argc和argv的使用。

发表于 2018-12-22 更新于 2020-01-10 分类于 C++ Valine：
本文字数： 551 阅读时长 ≈ 1 分钟

首先我们先编译该文件，gcc -o test test.c，得到test文件。

（如果执行gcc test.c，是得到a.out）

  • 如果我们在终端执行：./test

    则会报错：
    参数应该为两个

  • 如果我们在终端执行：./test Pineapple

    输出： 

    *第一个参数：./test*  
*第二个参数：Pineapple*

  • 如果我们需要输入的第二参数是Pineapple Catle，在终端执行：./test Pineapple Catle

    则会报错：
    参数应该为两个

    因为程序会以空格键区分各个参数。

  • 解决空格的问题，加上引号即可。在终端执行：./test “Pineapple Catle”

    输出：
    第一个参数：./test
    第二个参数：Pineapple Castle
